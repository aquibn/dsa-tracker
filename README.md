# DSA 30-Day Mastery Tracker

A powerful, standalone tracker designed to help you conquer **200 Data Structures & Algorithms problems in 30 days**. 

![DSA Tracker](https://img.shields.io/badge/Status-Active-success)
![React](https://img.shields.io/badge/React-18.2-blue)
![Vite](https://img.shields.io/badge/Vite-5.0-purple)

## 🚀 Features

- **AI Mastery Assistant**: Integrated Gemini AI to act as your personal coding coach:
  - **💡 Intuition**: Get high-level pattern explanations without spoilers.
  - **🔍 Review**: AI-powered analysis of your notes and complexity.
  - **🪜 Tiered Hints**: Progressively more detailed hints (DS -> Logic -> Pseudocode).
- **200 Problem Roadmap**: Carefully curated list of 200 problems organized into a 30-day schedule.
- **Progress Tracking**: Monitor your completion percentage and daily targets.
- **LeetCode Integration**: Direct links to solve every problem on LeetCode.
- **Bilingual DS Reference**: A built-in cheat sheet for all major Data Structures with snippets in both **C++ STL** and **Python**.
- **Spaced Repetition Ready**: Track your confidence level (Low, Medium, High) and keep revision notes for every problem.
- **Persistent Data**: Your progress is automatically saved to your browser's local storage.
- **Mac App Ready**: Configuration included for running as a native macOS application via Electron.

## 🛠️ Tech Stack

- **Frontend**: React (Hooks, Functional Components)
- **Tooling**: Vite for lightning-fast development
- **Desktop**: Electron (Native macOS integration)
- **Styling**: Modern Vanilla CSS

## 📋 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/aquibn/dsa-tracker.git
   cd dsa-tracker
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App
- **Web Mode**:
  ```bash
  npm run dev
  ```
  Then open [http://localhost:5173](http://localhost:5173) in your browser.

- **Desktop Mode (macOS)**:
  ```bash
  npm run electron-dev
  ```

## 📅 Roadmap Overview
- **Days 1-5**: Arrays, Hashing, Two Pointers, Sliding Window.
- **Days 6-10**: Stack, Binary Search, Linked List.
- **Days 11-20**: Trees, Heaps, Backtracking, Graphs.
- **Days 21-30**: Advanced Graphs, Dynamic Programming (1D/2D), Bit Manipulation, Math.
